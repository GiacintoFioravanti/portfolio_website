Real-Time Chat App

A full-stack real-time messaging app built with the MERN stack (MongoDB, Express, React, Node.js), Socket.io, and styled with TailwindCSS and Daisy UI. Features include JWT-based authentication and authorization, real-time messaging with Socket.io, online user status tracking, and global state management with Zustand.

MongoDB Express React Node.js Socket.io TailwindCSS 2025

Comic Book Collection System

In my software engineering design course, I collaborated with a team of four developers to build a comic book collection system. It organizes comic books from a CSV file into various categories like author, publisher, volume, and series. Users can create their own collections, stored in a PostgreSQL database.

Java PostgreSQL Team Project 2024

Nightclub Management System

In my Web Engineering course, I created a web app to help nightclubs keep track of customer numbers efficiently. It made sure they followed safety rules by connecting to a PostgreSQL database, storing maximum and current customer counts for each club.

Web Development PostgreSQL 2023

Engineered a versatile sonar system by integrating Arduino boards and ultrasonic sensors. The system was tailored for seamless attachment to diverse objects upon detection. This project was programmed using Arduino boards in C++ to accurately record distances and enable customized trigger responses upon object detection.
